# Framing Assistant

Guided mat and frame planning scaffold built in Expo + React Native.

## Commands

- `npm install`
- `npm run start`
- `npm run ios`
- `npm run android`
- `npm run typecheck`

## Notes

- The scaffold mirrors Darkroom Assistant's React Native stack and general UI language as closely as practical without copying product-specific features.
- The draft data model keeps `artworkSize`, `visibleReveal`, `matOpeningSize`, and `outerMatSize` as separate concepts from the start.
